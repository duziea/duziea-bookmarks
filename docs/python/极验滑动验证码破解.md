selenium 只能匀速滑动
为了模拟真人滑动，需要构建滑动轨迹 track,
实现先加速再减速，滑过了，再滑回。

```python
def get_track(self, distance):
        '''
        输入要滑动距离distance = end-start
        模拟滑动距离,
        先加速滑动后减速，滑过了，再滑回
        输出每次滑动的轨迹track
        '''
        v = 10
        a = 0
        t = 1
        track = []
        s = 0
        mid = distance / 2
        while s < distance:
            if s < mid:
                a = 6 * random.random()
            else:
                a = -3 * random.random()
            v0 = v
            v = v0 + a * t
            s0 = v0 * t + 1 / 2 * a * t * t
            s += abs(s0)
            track.append(s0)
        d = distance - s
        track.append(d)
        print(track)
        return track
```
